Motor turbo é a tecnologia dominante nos carros novos brasileiros — T270 da Stellantis, TSI da Volkswagen/Audi, TurboFlex da Fiat, motor turbo da GM no Onix Turbo. Todos exigem óleo de maior qualidade que um motor aspirado equivalente.

## Por que motor turbo degrada óleo mais rapidamente

O turbocompressor funciona com o calor do escapamento para comprimir o ar de admissão. Os mancais do turbo são lubrificados pelo óleo do motor — e ficam expostos a temperaturas de 400°C ou mais durante uso intenso.

Quando o motor é desligado abruptamente após trabalho pesado, o fluxo de óleo para. O turbo, ainda quente, "coze" o óleo nos mancais — formando verniz e depósitos que causam desgaste. Um motor que funciona com o turbo danificado pode falhar completamente.

## Identificando a especificação correta

### Viscosidade (ex: 5W-30, 0W-20)
- **W** = Winter — performance em temperatura baixa
- Número antes do W = fluidez no frio (menor = melhor partida no frio)
- Número após W = viscosidade em temperatura alta (maior = maior proteção em calor)

### API (American Petroleum Institute)
- **SN, SN Plus, SP** — padrões mais recentes para motores a gasolina
- Motor turbo deve usar no mínimo API SN; SP é o mais atual
- Nunca usar API SL ou anterior em turbo moderno

### ACEA (padrão europeu — relevante para TSI, TDI)
- **ACEA A3/B4** — proteção alta, viscosidade estável, para motores europeus de alta performance
- **ACEA C2/C3** — baixo SAPS (ceniza), compatível com catalisadores modernos

## Como escolher na prática

| Motor | Viscosidade | Classificação | Intervalo típico |
|-------|------------|--------------|-----------------|
| TSI 1.0T / 1.4T | 0W-20 ou 5W-30 | API SP / ACEA A5/B5 | 10.000-15.000 km |
| T270 1.3T (Jeep/Fiat) | 0W-20 | API SP, Stellantis MS-6395 | 10.000 km |
| Firefly 1.0T | 5W-30 | API SN Plus | 10.000 km |
| Motor 1.6T (Cruze/S10) | 5W-30 | API SP, Dexos Gen 2 | 10.000 km |
| Motor 2.0T (Corolla/RAV4) | 0W-16 | API SN / Toyota WS | 10.000 km |

**Regra de ouro:** Manual do proprietário determina o óleo correto. Qualquer dúvida, usar full sintético com a viscosidade correta e API SP ou superior.

## O que nunca fazer com motor turbo

1. **Nunca desligar imediatamente após uso intenso** — o turbo precisa esfriar com o motor girando
2. **Nunca usar óleo semi-sintético em turbo moderno** — não tem estabilidade de viscosidade adequada
3. **Nunca estender o intervalo além do recomendado** — óleo degradado destrói mancais do turbo
4. **Nunca ignorar baixo nível de óleo** — turbo depende de pressão de óleo constante

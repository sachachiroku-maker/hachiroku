O Tracker 1.0 turbo é o SUV mais vendido do Brasil — e o consumo de óleo é a dúvida mais frequente dos proprietários com mais de 30.000 km. A boa notícia: na maioria dos casos, o consumo está dentro do normal para motores turbo modernos.

## Por que motores turbo consomem mais óleo

Motores turbo têm temperaturas de trabalho mais altas que aspirados. O óleo 0W-20 especificado pelo CSS Prime é extremamente fluido — essa fluidez é necessária para lubrificar o turbo a frio rapidamente, mas também significa que ele evapora mais a altas temperaturas de operação.

Além disso, a GM (como outros fabricantes) dimensiona os anéis de pistão de motores modernos com tolerâncias muito estreitas para reduzir o atrito — menos atrito = menor consumo de combustível. A contrapartida é que esses anéis permitem um pequeníssimo vazamento de óleo para a câmara, especialmente em motores novos.

## O que é normal vs. o que não é

| Consumo | Avaliação |
|---------|-----------|
| 0-200 ml / 5.000 km | Normal — muito bom |
| 200-500 ml / 5.000 km | Normal para motor turbo |
| 500 ml - 1L / 5.000 km | Aceitável mas monitorar |
| >1L / 5.000 km | Investigar causa |

## Garantia GM para consumo de óleo

A GM cobre defeitos de consumo de óleo anormal dentro da garantia (3 anos / 100.000 km no Tracker). Para acionar a garantia, é necessário apresentar registro do consumo (recibos de compra de óleo, nível anotado com data/km). Sem documentação, a reclamação é difícil de processar.
